This folder is where the hero image belongs.

index.html currently points to: images/hero-ride.jpg
That file isn't included here — I wasn't able to download the real ride photo
into this package (my sandbox can't reach image-hosting domains directly).

To finish it yourself:
1. Save a ride/rideshare photo you like as: images/hero-ride.jpg
   (Recommended size: ~1200px wide, landscape orientation)
   A good free source: https://unsplash.com/s/photos/rideshare
2. That's it — index.html already references images/hero-ride.jpg, so it will
   pick it up automatically once the file exists.

Until you add it, the hero section will show a broken image icon.
